Hide and Seek with Visibility Constraints using Control Barrier Functions

Overview

This project explores the challenge of hide-and-seek games between autonomous agents with field-of-view (FoV) constraints. We propose a novel planning and control framework using Control Barrier Functions (CBFs) to ensure visibility while optimizing long-term costs. This research has applications in robotics, AI, and surveillance.

Key Features

  • Field-of-view (FoV) constraints modeling for realistic scenarios
  • Control Barrier Functions (CBF) for dynamic safety and visibility
  • Long-term cost optimization framework for strategic planning
  • Applicable to real-world robotic systems and AI agents
